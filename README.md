# FireKeeper Releases

**FireKeeper** — 日本語 × 安全な本番メンテナンスに特化した Mac 用 Firestore 管理アプリの配布用リポジトリです。

## ダウンロード

👉 **[最新版をダウンロード（macOS）](https://github.com/knitbow/firekeeper-releases/releases/latest/download/FireKeeper.dmg)**

すべてのバージョンは [Releases](https://github.com/knitbow/firekeeper-releases/releases) から。

## インストール方法

1. `FireKeeper.dmg` をダウンロードして開く
2. `FireKeeper.app` を `アプリケーション` フォルダにドラッグ
3. **ダブルクリックでそのまま起動できます**（Developer ID 署名 + Apple 公証済み）
